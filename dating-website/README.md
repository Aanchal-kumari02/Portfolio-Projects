# Dating Website

A modern dating website with interactive profile cards and swipe functionality.

## Features

- Interactive profile cards with smooth animations
- Swipe functionality (like, reject, super like)
- Match percentage display
- Beautiful gradient UI
- Smooth transitions and animations
- Responsive design

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
