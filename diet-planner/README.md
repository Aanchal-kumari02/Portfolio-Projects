# Diet Planner

A modern diet planner application with interactive meal planning and nutrition tracking.

## Features

- Daily nutrition tracking (calories, protein, carbs, fats)
- Meal planning for breakfast, lunch, dinner, and snacks
- Water intake tracker
- Interactive UI with smooth animations
- Progress bars with visual feedback
- Beautiful gradient design
- Responsive layout

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
