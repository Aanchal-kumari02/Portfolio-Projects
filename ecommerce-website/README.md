# Ecommerce Website

A modern ecommerce website with interactive UI and smooth transitions.

## Features

- Beautiful, modern UI with gradient effects
- Smooth animations and transitions
- Product catalog with search functionality
- Shopping cart and wishlist
- Responsive design
- Interactive product cards with hover effects

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
